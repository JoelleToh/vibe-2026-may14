
# Project Blueprint

## Overview

This project is a simple web application that provides a "Toto" random number generator. The application is built using HTML, CSS, and JavaScript, and it is designed to be a simple, single-page application.

## Current Features

*   **Toto Random Number Generator:**
    *   Generates 6 unique random numbers between 1 and 49.
    *   Displays the generated numbers in a clear and easy-to-read format.
    *   A "Generate" button to trigger the number generation.
*   **Day/Night Mode:**
    *   Toggle between "Day" (light) and "Night" (dark) themes.
    *   Theme preference is persisted in the browser's `localStorage`.
    *   Initial theme is detected based on system preferences if no saved preference exists.
    *   Smooth transitions between theme switches.

## Design and Style

*   **Layout:** A simple, centered layout with a responsive container for the numbers.
*   **Typography:** Clean and readable sans-serif fonts.
*   **Color Scheme:**
    *   **Day Mode:** Light gray background with dark gray text and blue buttons.
    *   **Night Mode:** Dark gray background with light gray text and purple buttons.
*   **Interactivity:**
    *   The "Generate" and "Theme Toggle" buttons have clear hover states and smooth transitions.
    *   Transitions are applied to background and text colors for a polished feel.
